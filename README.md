# AI Finance Portfolio Manager

## Overview

The AI Finance Portfolio Manager is a project aimed at helping users manage their investment portfolios using Artificial Intelligence techniques. This application analyzes various financial metrics and market trends to provide insights and recommendations for optimizing investment decisions.

## Features
- **Portfolio Tracking**: Keep track of your investments in real-time.
- **Market Analysis**: AI-driven market analysis to suggest potential investments.
- **Performance Reports**: Detailed reports on investment performance over time.
- **User-Friendly Interface**: An intuitive interface for easy navigation and management.

## Installation

To install the project, clone the repository:
```bash
git clone https://github.com/esingh16/ai-finance-portfolio-manager.git
```

Change into the project directory:
```bash
cd ai-finance-portfolio-manager
```

Then install the necessary dependencies:
```bash
pip install -r requirements.txt
```

## Usage

To run the application, use the following command:
```bash
python main.py
```
